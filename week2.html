Week 02

Deep Dive into AWS Services

February 10 - 14, 2025

The first week covered AWS-CDK fundamentals, including project setup, key concepts, and hands-on labs. Participants learned about infrastructure as code, debugging, and best practices for managing cloud resources. They also worked with AWS services such as Cognito for authentication, Lambda for serverless computing, DynamoDB for NoSQL databases, and SQS for message queuing. The week concluded with a hands-on lab where trainees built and deployed applications using AWS-CDK.

As part of our onboarding at StraStan Solutions Corp, each trainee was assigned API Gateway endpoints to integrate with AWS Lambda, DynamoDB, and other AWS services.

Understanding the Task

This week, I worked on managing sections and subcategories in the Marketplace module and handling card creation and reviews in the Cards module. These tasks required integrating API Gateway, Lambda functions, DynamoDB, SQS queues, Cognito authentication, and EventBridge for event-driven processing. In the Marketplace module, I implemented endpoints for organizing sections and setting display rules. The POST /admin/sections/organize endpoint, handled by the organizeContent Lambda function, structures content dynamically and stores data in the Sections table. The POST /admin/sections/display endpoint, powered by the setDisplayRules function, manages visibility rules for sections. Additionally, I developed the POST /admin/subcategories endpoint, which allows subcategory creation under specific categories, ensuring proper hierarchy in the Subcategories table.

In the Cards module, I implemented the POST /cards endpoint, using the createCard Lambda function to store metadata in the Cards table. I also worked on the POST /admin/listings/review endpoint, which allows admins to review listings through the reviewListing function, ensuring that status updates are processed efficiently.

Throughout development, I utilized AWS Lambda for backend logic, DynamoDB for structured storage, and API Gateway for seamless communication. SQS handled task queuing, Cognito secured authentication, and EventBridge ensured smooth event tracking. By the end of the week, I aim to deepen my understanding of event-driven workflows and API-driven architecture while enhancing system efficiency. This hands-on experience is valuable for my ongoing work at StraStan Solutions Corps.
